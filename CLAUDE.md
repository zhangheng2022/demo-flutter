## 项目概览

这是一个功能完整的 Flutter 相机应用，支持多个平台（Android、iOS、macOS、Windows、Linux、Web）。

**关键信息：**
- Dart SDK: ^3.11.1
- Flutter 框架，使用 Material Design 3
- 使用 Riverpod 进行状态管理
- 使用 go_router 进行页面导航
- 使用 camera 插件实现相机功能
- 使用 flutter_lints 进行代码质量检查
- 页面内容优先使用中文
